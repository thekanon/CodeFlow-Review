# 연속 코드 리뷰 도우미 🔍🧠

ChatGPT와 연동하여 코드를 자동으로 연속 리뷰해주는 크롬 확장 프로그램입니다.  
입력된 코드에 대해 일련의 코드 품질 질문을 자동으로 ChatGPT에 보내고, 각 응답에서 리팩토링된 코드를 추출하여 다음 질문에 반영합니다.  
최종적으로 개선된 버전의 코드와 요약까지 자동으로 받아볼 수 있습니다!

---

## 📦 기능 소개

- ✅ 코드에 대한 주석 평가 및 보완
- 🔄 순수 컴포넌트 원칙 준수 여부 점검
- 🛠 추가 개선점 제안 및 리팩토링
- 📝 최종 요약 결과를 마크다운 형식으로 출력

---

## 🧩 설치 방법

1. 이 저장소를 클론하거나 ZIP으로 다운로드합니다.
2. 크롬 브라우저에서 `chrome://extensions` 페이지로 이동합니다.
3. 우측 상단의 **"개발자 모드"**를 활성화합니다.
4. **"압축 해제된 확장 프로그램을 로드"** 버튼을 클릭하고 이 폴더를 선택합니다.

---

## 🚀 사용 방법

1. [https://chatgpt.com](https://chatgpt.com) 에 접속합니다.
2. 확장 프로그램 팝업을 열고, 리뷰하고자 하는 코드를 입력한 뒤 **"리뷰 시작"** 버튼을 클릭합니다.
3. ChatGPT 창에서 자동으로 질문이 순차적으로 입력되며, 응답과 함께 리팩토링된 코드가 반영됩니다.
4. 모든 질문 완료 시 알림이 표시됩니다.

---

## 🛠 개발자 정보

- **Manifest Version**: 3
- **콘텐츠 스크립트**를 통해 ChatGPT 페이지에 직접 접근
- 질문 로직은 `injectedStartReviewProcess` 함수 내에 정의되어 있습니다.
- 리팩토링된 코드는 ChatGPT 응답에서 자동으로 추출하여 다음 질문에 반영됩니다.

## ⚠️ 주의사항

- ChatGPT 페이지 구조가 변경될 경우 셀렉터가 동작하지 않을 수 있습니다. 그럴 경우 `content.js` 내 셀렉터를 수정해야 합니다.
- ChatGPT 응답 시간에 따라 자동화 진행 속도가 달라질 수 있습니다.
